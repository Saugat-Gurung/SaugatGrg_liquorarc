# 📱 LiquorArc: Native Android E-Commerce Platform

## Project Overview
LiquorArc is a fully hosted, native Android e-commerce application developed as a comprehensive Bachelor's capstone project. Designed specifically for the Nepalese market, it provides a seamless mobile shopping experience for retail, complete with a robust web-based administrative backend for real-time inventory and order management.

## Technical Architecture
* **Platform:** Android (Native)
* **Language:** Strict Object-Oriented Java
* **Architecture:** Client-Server model with custom API integration
* **UI/UX:** Custom XML layouts designed for intuitive mobile navigation

## Core Features
* **Native Android Frontend:** Built entirely in Java, prioritizing high performance and a smooth, responsive user interface without reliance on cross-platform frameworks.
* **API-Driven Communication:** All application data—from product catalogs to user sessions—is dynamically handled via API calls to the central server.
* **Web-Based Admin Panel:** A dedicated web dashboard utilized for managing the product lifecycle, updating stock levels, and securely handling client-server connection states.
* **Full E-Commerce Lifecycle:** Supports the complete user journey from product browsing and selection to secure ordering.

## Engineering Highlights
This application was built entirely from the ground up utilizing foundational software engineering principles. By manually writing the API connections and server interactions in strict Java, this project established a deep understanding of asynchronous network requests, mobile state management, and secure client-server data transfer long before the era of AI-assisted coding.
